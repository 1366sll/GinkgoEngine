## 1. 整体布局：三栏式工作台

打开编辑器，你会看到三个竖着的区域：

```
+----------------+------------------------+------------------+
|  左侧：资源库   |   中间上：场景列表      |  右侧：场景编辑   |
|                |                        |                  |
|  立绘          |  [索引][说话人][文本].. |  ⏪撤销 ⏩重做    |
|  背景          |  0  林辰  明天就是...  |  说话人_________ |
|  音乐          |  1  文小管 你好...     |  对话文本_______ |
|  音效          |                        |  [🤖 AI改写]     |
|  CG            |   中间下：实时预览      |  背景键_________ |
|  语音          |  +------------------+  |  BGM____________ |
|  视频          |  | 游戏画面预览      |  |  ...更多字段     |
|  ImgBB图床     |  | 对话框预览        |  |                 |
|                |  | [🔄] [🎞️] [🗺️]   |  |  更新场景      |
|                |  +------------------+  |  导出项目      |
+----------------+------------------------+ 🔍诊断 📦分章导出 |
                                  +--------+-----------------+
                                  | 🎭 角色设定档案          |
                                  +--------------------------+
```

- **左侧**：管理所有游戏资源 + ImgBB 图床上传
- **中间上**：场景列表表格，可以搜索、跳转、增删场景
- **中间下**：实时预览当前场景 + 流程图按钮
- **右侧**：编辑当前选中场景的所有属性 + 角色设定档案

## 2. 左侧：资源库 —— 你的素材仓库

左侧从上到下有八个资源分类：

| 类别 | 说明 |
|------|------|
| 🎨 立绘 (sprites) | 角色的各种表情差分 |
| 🌄 背景 (bg) | 每个场景的背景画面 |
| 🎵 音乐 (bgm) | 循环播放的背景音乐 |
| 🔊 音效 (se) | 短音效：敲门声、手机铃声等 |
| 🖼️ CG (cg) | 特殊事件的全屏大图 |
| 🎙️ 语音 (voice) | 配合对话播放的配音 |
| 🎬 视频 (video) | 过场动画等 |
| ☁️ ImgBB 图床 | 云端图片托管（详见第21节） |

### 如何添加素材？

点击任意类别下的 **"+ 上传XXX"** 按钮，选择你电脑上的文件。系统会让你给这个文件起一个**键名**（key），比如：

- 立绘：`normal`、`shy_happy`、`serious`
- 背景：`classroom`、`dorm`
- 音乐：`main`、`sad_theme`

> **提示**：键名是你在剧本里引用素材时使用的名字，建议用英文小写 + 下划线，简短好记。

### 如何预览素材？

直接**点击**资源列表里的资源名称即可弹出预览窗口。音乐和音效会播放试听，视频会弹出播放器。

### 如何替换素材？

每个资源旁边都有一个**"替换"**按钮，点它选择新文件就能换掉（键名保持不变）。

## 3. 中间上方：剧本场景列表 —— 故事的大纲

这里以表格形式展示你的全部场景，每一行就是一个场景。

### 表格各列含义

| 列名 | 内容 |
|------|------|
| 索引 | 场景编号，从 0 开始 |
| 说话人 | 这句话是谁说的 |
| 文本(简) | 对话内容的前 30 个字 |
| 背景 | 当前使用的背景图键名 |
| 立绘 | 角色立绘（多立绘逗号分隔，🎥 有视频，🧹 清场） |
| CG | 关联的全屏 CG（🎬 有过渡动画） |
| 分支 | 分支选项数量（🔀 条件跳转） |
| 副标题 | 章节副标题（仅转场场景有效） |

### 操作方法

- **点击某一行** → 选中它，右侧详情区自动加载
- **"+ 新增场景"** → 在末尾追加一个空白场景（`Ctrl+N`）
- **"在当前位置前插入"** → 在当前场景前插入（`Ctrl+Shift+N`）
- **"删除选中"** → 删除当前场景（至少保留一个）（`Delete`）
- **"定位到下一个章节"** → 跳到下一个标记了章节转场的场景

### 搜索功能

在搜索栏输入关键词（说话人、对话内容、背景、立绘、地点等）：
- **"查找下一个"** — 从当前场景往后搜
- **"上一个"** — 从当前场景往前搜
- 按回车键 = 查找下一个，`Ctrl+F` 聚焦搜索框

搜索会循环——搜到末尾会从头继续。

## 4. 中间下方：游戏内预览 —— 所见即所得

选中一个场景后，预览区实时显示游戏中的样子：背景图、立绘、说话人、对话文字、分支选项按钮、好感度条、画面效果、地点提示、CG/视频/过渡/变量摘要信息。

三个按钮：
- **"🔄 刷新预览"** — 手动重新渲染（`Ctrl+R`）
- **"🎞️ 预览CG"** — 新标签页打开 CG 原图
- **"🗺️ 路线流程图"** — 打开交互式流程图面板（详见第16节）

## 5. 右侧：场景详情编辑器

选中一个场景后，所有表单字段加载该场景数据。

### 核心字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 说话人 | 这句话的说话人 | `林辰`、`文小管` |
| 对话文本 | 正文（可用 BBCode） | `你好，我是……` |
| 背景键 | 背景图键名 | `classroom` |
| BGM键 | 背景音乐键名 | `main` |
| 下一个索引 | 跳转目标（-1=结束） | `5`、`-1` |

### 立绘配置

| 字段 | 说明 |
|------|------|
| 多立绘（sprites数组） | 多角色同屏（详见第7节），留空用旧版单选 |
| 立绘键（单选） | 单角色立绘，多立绘为空时生效 |

### 场景控制

| 字段 | 说明 |
|------|------|
| 地点提示 | 左上角地点名，如 `教学楼` |
| 清场 (clearStage) | 是=所有立绘滑出消失 |
| 章节转场 | true/false 是否切章节 |
| 章节副标题 | 章节名（转场=true时显示） |

### 特殊功能

| 字段 | 说明 |
|------|------|
| 场景过渡效果 | 切换动画（详见第11节） |
| 插入视频 (videoKey) | 播一段视频 |
| 变量赋值 (setVar) | 设置游戏变量（详见第10节） |
| 条件跳转 (conditionNext) | 根据变量自动分支（详见第10节） |
| 插入CG (cgKey) | 全屏 CG 插画 |
| 画面演出效果 | 滤镜/震动/白闪（详见第8节） |
| 分支选项 | 是否含玩家选择（详见第9节） |

### 结局与好感度

| 字段 | 说明 |
|------|------|
| 结局键 (endingKey) | 结局标识符，如 `good_ending` |
| 结局标题 (endingTitle) | 结局标题，如 `温柔的再见` |
| 好感度 (affinity) | 0-100，-1=隐藏条 |
| 好感度标签 | 条标签，如 `文小管好感度` |

### 操作按钮

| 按钮 | 说明 |
|------|------|
| ⏪ 撤销 | 回退上一步（`Ctrl+Z`），详见第18节 |
| ⏩ 重做 | 恢复撤销（`Ctrl+Y`），详见第18节 |
| 💾 更新场景 | 保存编辑（`Ctrl+S`） |
| 📦 导出完整项目 | 导出 JSON（`Ctrl+E`） |
| 🔍 项目诊断 | 检查问题，详见第17节 |
| 📂 导入完整项目 | 从 JSON 恢复 |
| 🔄 重置默认数据 | 恢复示例数据 |
| 📦 分章导出 | 按章节分割导出，详见第20节 |

## 6. BBCode：给文字加点魔法

### 颜色文字

```
[c=red]红色[/c] [c=blue]蓝色[/c] [c=gold]金色[/c] [c=green]绿色[/c] [c=pink]粉色[/c]
```

### 样式

```
[i]斜体[/i]  [b]粗体[/b]
```

### 动画特效

```
[shake]震动文字[/shake]
```

### 字号

```
[big]大号强调[/big]  [small]小声嘀咕[/small]
```

文本区上方有一排彩色按钮，选中文字后点击即可自动包裹标签。

## 8. 画面演出效果

### 一次性滤镜
怀旧(sepia) / 黑白(grayscale) / 模糊(blur) / 增亮(bright) / 变暗(dark) / 高饱和(saturate-high) / 低饱和(saturate-low)

### 瞬间触发
震动(shake) / 白闪(flash) / 画面推进(zoom-in)

### 持续氛围
暖色调(mood-warm) / 冷色调(mood-cool) / 戏剧化(mood-dramatic) / 暗角(vignette) / 电影黑条(letterbox)

### 清理
勾选"清除所有效果"一键移除。

持续时间填 `0`=持久，填毫秒数（如 `3000`=3秒）为一次性。

## 10. 条件跳转与变量系统（进阶）

### 变量赋值 (setVar)

点击 **"+ 添加变量"** 添加键值对：
- 数字值自动存为 number 类型，其他存为 string
- 在对话中用 `{变量名}` 引用：`这是我们第{chapter}次见面了呢。`

### 条件跳转 (conditionNext)

支持 `>=` `<=` `>` `<` `==` `!=` 运算符：
- `affection>=50` → 跳转到场景 8
- `metTeacher==true` → 跳转到场景 12
- `default` → 都不满足时的默认跳转

按添加顺序从上到下检查，第一个匹配生效。

## 14. AI 协同写作（新功能）

编辑器集成 DeepSeek AI 帮你润色对白。

### 使用步骤
1. 在"对话文本"中输入对白
2. 选择改写风格下拉框：**更温柔 / 更活泼 / 更文艺 / 更简洁 / 更深情**
3. 点击 **"🤖 AI 改写"** 按钮
4. AI 返回改写结果，自动填入文本框

### 前提条件
需要 DeepSeek API Key（与后日谈共用）。未设置时会弹出提示。

### 技巧
- 改写保留原意和角色身份
- 不满意可再次点击改写
- 配合 **Ctrl+Z** 随时回退原文

## 20. 导入、导出与保存

### 完整项目导出
`Ctrl+E` 导出单个 JSON（场景 + 资源 + 角色档案 + 图床映射）。

### 分章导出（新增）
点击"📦 分章导出"，按章节分割成：
- `galgame_index.json` — 索引文件
- `chapter_0.json`、`chapter_1.json`… — 各章节场景

**优势**：游戏引擎自动启用**懒加载模式**——先加载序章即可开始，后续章节接近时后台自动加载，大幅缩短首屏等待。

### 导入
选择 JSON 文件导入（覆盖当前数据，请先备份）。

### 在游戏中加载
导出 JSON → 打开 `gal(2).html` → 标题画面"加载项目文件" → 选择文件。

## 21. ImgBB 图床上传

左侧底部的"☁️ ImgBB 图床"面板：

## 23. 常见问题

### Q: 立绘不显示？
A: 检查资源库键名是否一致、多立绘是否误填导致单选被忽略。

### Q: 误删了场景？
A: 按 **Ctrl+Z** 撤销，支持最近 50 步。

### Q: AI 改写没反应？
A: 需要 DeepSeek API Key（与后日谈共用），未设置时会弹窗提示。

### Q: 分支选项点了没反应？
A: 检查"是否含有分支选项"选了"是"，每个选项的跳转索引有效。

### Q: 条件跳转不生效？
A: 表达式格式正确（无空格）、变量名与 setVar 一致、条件按顺序检查。

### Q: 导出后游戏报错？
A: 先点"项目诊断"查看具体问题（越界跳转、缺失资源引用）。

### Q: 分章导出 vs 完整导出？
A: 完整导出=一个大 JSON，适合短篇。分章导出=多个小 JSON，游戏引擎懒加载模式，适合长篇（场景 > 200 推荐）。
