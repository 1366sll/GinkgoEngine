## 1. 启动与主菜单

打开 `gal(2).html`（推荐 Chrome 或 Edge），看到主菜单：

```
+----------------------------------+
|         银杏落雨时               |
|                                  |
|   [开始游戏]  [设置]             |
|   [鉴赏]      [后日谈]（解锁后） |
|   [结束游戏]                     |
+----------------------------------+
```

| 按钮 | 作用 |
|------|------|
| **开始游戏** | 新游戏 / 读档 / 章节选择 |
| **设置** | 音量、速度、热键、项目管理 |
| **鉴赏** | CG、立绘、音乐、结局、成就、路线图 |
| **后日谈** | AI 角色对话（通关任意结局后解锁） |
| **结束游戏** | 关闭页面 |

> 如果之前玩过，"继续游戏"按钮可直接从上次中断处继续。

## 2. 进行游戏 —— 阅读剧情

### 画面布局

```
+----------------------------------+
|  地点提示（左上角）               |
|     背景图片                     |
|         [立绘]                   |
|                                  |
|  对话区（底部）                   |
|  [说话人]                        |
|  对话文字..............................|
|  [快进] [已读跳过] [自动] [...] [继续] |
+----------------------------------+
```

### 操作方式

- **鼠标左键** / **空格** / **Enter** → 下一句
- **↑ 键** → 回溯上一句（需在设置中开启）
- **数字键 1/2/3** → 快速选择分支选项

### 快进与自动模式

| 按钮 | 效果 |
|------|------|
| **快进** | 快速显示对话（不跳过选项和CG） |
| **已读跳过** | 只跳过已读文本，新文本正常显示 |
| **自动** | 自动播放，句间停顿后可调整 |

## 3. 存档与读档

### 手动存档
- 底部工具栏 **"快速存档"** 按钮
- 按 **F5** 或 **Q** 快速存档
- 在存档页面可选 10 个存档位

### 自动存档
- 每个章节切换时自动保存
- 主菜单"继续游戏"读取最新自动存档

### 存档存储位置
存档保存在浏览器本地存储中。清除缓存会丢失存档。

### 加载项目文件
主菜单 **"设置 → 项目管理 → 加载项目文件"** 选择 `.json` 文件加载剧本。

## 4. 存档分享码（新功能）

在存档页面点击 **"分享"** 和 **"导入"** 按钮，可以在玩家之间传递游戏状态。

### 分享你的进度

1. 在游戏中打开存档页面
2. 点击 **"分享"** 按钮
3. 弹窗显示分享码 + QR 码
4. 复制分享码发送给朋友，或让对方扫码

### 导入别人的进度

1. 在存档页面点击 **"导入"** 按钮
2. 粘贴别人分享的分享码
3. 点击"跳转"——游戏会跳转到对方所在的确切场景和状态

### 分享码包含什么

- 当前场景位置
- 所有游戏变量（好感度、flag 等）
- 当前 BGM
- 文件大小约 200-500 字符，便于复制分享

### 右键菜单

游戏中**鼠标右键** → 选择"分享当前状态"也可以快速分享。

## 5. 设置与个性化

主菜单"设置"或游戏内右键 → 设置：

### 音量

| 选项 | 作用 |
|------|------|
| BGM音量 | 背景音乐大小 |
| 音效音量 | 效果音大小 |
| 语音音量 | 角色配音大小 |
| 语音闪避强度 | 说话时背景音乐降低的程度 |

### 播放行为

| 选项 | 作用 |
|------|------|
| 文字速度 | 打字效果的速度（15~120 ms/字） |
| **自适应速度** ⭐新增 | 根据你的阅读习惯自动调整文字速度（详见下方） |
| 自动播放速度 | 自动模式下每字停顿时间 |
| 最小间隔 | 自动播放句间最小等待时间 |

### 自适应阅读速度（新功能）

开启后（默认开启），引擎会学习你的阅读节奏，自动调整文字显示速度：

- 你点"继续"越快，文字显示越快
- 你阅读越慢，文字显示越慢
- 调整范围：15~120 ms/字
- 当你手动拖动文字速度滑块时，自动学习暂停
- 可在设置中随时关闭

### 立绘切换

- **交叉淡化** → 立绘变化有淡入淡出
- **直接切换** → 立绘瞬间变化

### 滚轮回溯

- 开启后可用 **↑ 键** 回看上一句
- 可调节回溯动画速度

## 6. 鉴赏模式 —— 画廊

主菜单点击 **"鉴赏"** 进入，可查看已解锁内容：

| 选项卡 | 内容 |
|--------|------|
| **CG** | 全屏插画 |
| **人物立绘** | 所有表情差分 |
| **背景** | 场景背景 |
| **音乐** | BGM 试听 |
| **结局** | 已打出的结局列表 |
| **成就** | 获得的成就 |
| **路线图** | 交互式剧情分支流程图 |

### 如何解锁
- CG：剧情中第一次出现时解锁
- 立绘：第一次显示时解锁
- 音乐：第一次播放时解锁
- 结局：达到对应结局时解锁

### 封面设置
在鉴赏模式中可自定义主菜单的封面图和音乐：最近CG / 候选CG随机 / 全随机。

## 7. 后日谈 —— AI 角色对话（新功能）

通关任意结局后，主菜单出现 **"后日谈"** 按钮。

### 这是什么？

后日谈是主线剧情结束后的自由对话模式。你扮演主角**林辰**，与角色**文小管**通过 AI 实时生成对话。角色有完整的人格设定（性格、背景故事、说话风格），回复会带有情绪驱动的立绘/画面演出。

### 结局感知

后日谈会知道你通关的是哪个结局，AI 角色会呈现不同的态度和记忆：

- **好结局** → 她在感激和平静中，有些话想确认
- **普通结局** → 生活回到正轨，偶尔想如果选了不同选项会怎样  
- **真结局** → 释怀中带着新的力量，开始相信未来
- 不同结局下，同一个话题可能得到完全不同的回应

### 对话功能

| 功能 | 说明 |
|------|------|
| 打字输入 | 在底部输入框打字，按回车发送 |
| 快捷话题 | 点击 💬 按钮选预设话题 |
| 重新生成 | 对 AI 的回复不满意？点 🔄 重新生成 |
| 导出对话 | 点 📥 导出对话记录（同时下载 TXT + 精美 HTML） |
| 语音朗读 | AI 回复会自动用语音朗读（可关闭） |

### 画面效果

AI 回复中包含 BBCode 标签时，会触发画面演出：
- `[shake]` 画面震动、`[flash]` 白闪
- `[sepia]` 怀旧滤镜、`[warm]` 暖色调
- 角色立绘根据情绪自动切换表情
- 支持 Live2D 动态角色模型

### 设置面板

点击 ⚙️ 可调节：
- 打字速度
- BGM / 语音音量
- Live2D / 静态立绘切换
- DeepSeek API Key 管理

### API Key

后日谈需要 DeepSeek API Key。在设置面板中输入 `sk-...` 格式的 Key。Key 仅保存在本地浏览器中，使用浏览器指纹加密。

### 多角色未来扩展

架构已预留多角色支持。当编辑器导出包含多个角色档案的 JSON 后，后日谈可加载并切换对话对象。

## 9. 手柄支持

|------|------|
| **A / 叉** | 继续对话 / 确认 |
| **B / 圈** | 返回 / 菜单 |

| **Y / 三角** | 快进 |

## 10. 渐进式预加载（新功能）

加载项目后，游戏引擎会自动执行分层预加载：

```
Phase 0 → 封面图 + 封面BGM（立即显示）
Phase 1 → 场景背景 → 立绘 → CG → 背景音乐（按出场顺序）
Phase 2 → 音效 + 语音（后台闲时）
```

### 这意味着什么

- **标题画面秒开**：封面图和音乐最先加载
- **首句对白无白屏**：第一个场景所需的 BG 和立绘在标题画面停留期间就已缓存
- **进度可见**：左下角显示"预加载 XX%"进度
- **不阻塞游玩**：点"新游戏"时预加载自动中止

## 11. 懒加载 / 分章项目（新功能）

如果你的剧本是通过编辑器**分章导出**的（多个 JSON 文件），游戏引擎会自动启用懒加载模式：

- 先加载第 0 章（序章）→ 即可开始游戏
- 后续章节在玩家接近时自动后台下载
- 首屏加载时间大幅缩短

## 12. 常见问题

### Q: 游戏卡住了，点哪里都没反应？
A: 按 **Esc** 关闭弹窗，或刷新页面（会丢失未保存进度）。

### Q: 没有声音？
A: 检查浏览器是否静音（标签页喇叭图标），游戏设置中音量是否被调为 0。

### Q: 存档丢失了？
A: 存档在浏览器本地存储中。清除缓存或换电脑会丢失。可以用**存档分享码**备份关键进度。

### Q: 后日谈怎么解锁？
A: 通关任意结局后自动解锁。也可在开发者控制台（按 ` 键）输入 `epilogue` 手动解锁。

### Q: 后日谈输入后没反应？
A: 检查 DeepSeek API Key 是否已正确设置（设置面板 → API Key）。Key 以 `sk-` 开头。

### Q: 如何加载别人分享的剧本？
A: 主菜单 → 设置 → 项目管理 → 加载项目文件 → 选择 `.json` 文件。

### Q: 如何加载分享码？
A: 在游戏中的存档页面 → 点击"导入"按钮 → 粘贴分享码 → 跳转。

### Q: 自适应速度不生效？
A: 检查设置中的"自适应速度"开关是否开启（默认开启）。手动拖动速度滑块后自动学习不会立即重置——关闭再重新开启可以重置学习。

### Q: 手柄没反应？
A: 确认手柄已连接并被浏览器识别。可能需要先点击页面任意位置激活手柄支持。

### Q: 全屏后鼠标不见了？
A: 移动鼠标到屏幕底部，鼠标会自动出现。按 **F11** 退出全屏。
